# Group-M25M19012-M25M19015-_Dataset-Taxi-_UCU_BDA_Exam_Easter2026
Offline-First Taxi Demand Forecasting with PySpark &amp; Spark MLlib.  Predicts next-hour NYC taxi demand by zone for fleet optimization.  Local batch pipeline processes 3M+ trips. Random Forest model: R²=0.921.  Reproducible, no cloud infrastructure needed. Enables strategic driver  allocation before peak demand.
